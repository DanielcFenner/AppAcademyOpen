HTML Tags Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page should only have 1 `h1` element

Paragraphs

Paragraphs of text go in `p` tags

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in an `li` tag

Types of lists:

  • Unordered lists `ul`
  • Ordered lists `ol`

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in `li` tags.

Anchors (links)

Anchor tags create links to other pages. The URL goes in the `href` attribute Go to DuckDuckGo to search for more information

Setting `target="_blank"` will open the link in a new window See MDN for more details.

Anchor tags can also create links to a new location on the same page by referencing the id tag Jump to the top of the page.

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within it

Bold

You can also use the `strong` element to emphasize text.

Organizing content with `div` and `span`

`div`

The `div` element is a block element, so each `div` will show up on a new line
